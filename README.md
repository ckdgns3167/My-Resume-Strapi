## REST API 서버 Hosting 하기

-----------

- URL: https://myresumestrapi.herokuapp.com/

### ❓ 목적

- React로 나를 소개하는 résumé 사이트를 구현하던 중에, 나의 정보들을 API로 받아서 처리하면 어떨까? 하여 만들게 됨. 


### ⚡ 사용한 기술

- NodeJS
- **Strapi**: Node 패키지이며, REST API 서버 구축과 관리를 도움.
- **MongoDB Atlas**: 서버 데이터를 저장.
- **Heroku**: 서버 배포.
- **AWS s3**: 이미지 저장
- Git
